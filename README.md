# dotPLC
The SLMP-compatible software tool represents an actual PLC.

## Overview
dotPLC is SLMP-compatible software tool represents an actual PLC, that helps engineers test communication with external devices (e.g. HMI, PLC, Scada, Cognex Camera, PC ...) fast and easy.
